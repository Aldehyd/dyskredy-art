Raccourcis clavier

Ce site implémente divers composants complexes dont l'utilisation via le clavier est détaillé ci-dessous.

Comment naviguer au clavier au sein de cette fenêtre ?

Les informations sont regroupées dans un composant de type accordéon.
Utiliser la touche 'Tab' pour naviguer entre les en-têtes.
Lorsque le focus est positionné sur un en-tête, appuyer sur 'Entrée' ou 'Espace' pour dérouler le panneau correspondant (lorsqu'un nouveau panneau s'ouvre, les autres se ferment automatiquement).
Appuyer sur la touche 'Echap' pour fermer cette fenêtre.

Note : il est possible d'ouvrir directement cette fenêtre depuis n'importe quelle page du site en appuyant simultanément sur 'Control' + 'Alt' + 'S'.

  • Les fenêtres modales peuvent être fermées avec la touche 'Echap'.
    Le focus forme une boucle et ne sort pas de la fenêtre tant que cette dernière n'est pas fermée.

  • Utiliser les flèches directionnelles pour manipuler la valeur du potentiomètre.
    Les touches 'Flèche haute' et 'Flèche droite' permettent d'augmenter la valeur.
    Les touches 'Flèche basse' et 'Flèche gauche' permettent de diminuer la valeur.
    La touche 'Haut de page' permet d'augmenter la valeur plus rapidement qu'avec les flèches directionnelles.
    La touche 'Bas de page' permet de diminuer la valeur plus rapidement qu'avec les flèches directionnelles.
    La touche 'Fin' permet de positionner le potentiomètre sur sa valeur maximale.
    La touche 'Début' permet de positionner le potentiomètre sur sa valeur minimale.

  • Appuyer sur la touche 'Entrée' ou 'Echap' pour ouvrir la page correspondant au bouton survolé.
    Utiliser les touches directionnelles pour se déplacer dans le menu principal.
    Les touches 'Flèche droite' et ' Flèche gauche' permettent de se déplacer entre les boutons principaux en formant une boucle.
    Certains boutons principaux contiennent des sous-menus. Pour les dérouler, appuyer sur 'Flèche basse' ou 'Flèche haute'.
    'Flèche basse' déroule le sous-menu et positionne le focus sur le premier élément.
    'Flèche haute' déroule le sous-menu et positionne le focus sur le dernier élément.
    Les sous-menu peuvent être fermés en appuyant sur 'Echap'.
    Utiliser les touches 'Flèche haute' et 'Flèche basse' pour se déplacer à l'intérieur d'un sous-menu. Le déplacement forme une boucle de sorte que le focus ne sort pas du sous-menu.
    Par défault, les sous-menu ne s'ouvrent pas automatiquement lors de la navigation entre les boutons principaux du menu.
    Cependant, lorsqu'un sous-menu a été ouvert et qu'on utilise 'Flèche droite' ou 'Flèche gauche' pour naviguer entre les boutons principaux, les sous-menus s'ouvrent alors automatiquement au focus de leurs boutons parents.
    Pour désactiver l'ouverture automatique des sous-menus, fermer un sous-menu manuellement avec la touche 'Echap'.
    Il est possible d'atteindre chaque bouton du menu en pressant la lettre du clavier correspondant à la première lettre de son nom.
    Par exemple, pour atteindre le bouton 'Contact', appuyer sur la touche 'c'.
    Lors de la navigation dans un sous-menu, il est possible d'atteindre les liens de la même façon.

  • Le menu de changement de langue peut être déroulé avec les touches 'Entrée' et 'Espace' ou avec les touches 'Flèche haute' et 'Flèche basse'.
    Lorsque le menu n'est pas déroulé, 'Flèche basse' le déroule et place le focus sur le premier élément de la liste.
    Lorsque le menu n'est pas déroulé, 'Flèche haute' le déroule et place le focus sur le dernier élément de la liste.
    Le menu peut être fermé en appuyant sur la touche 'Echap'.
    Le déplacement à l'intérieur de la liste de choix se fait via les touches 'Flèche haute' et 'Flèche basse'. Elles forment une boucle de sorte que le focus ne sort jamais de la liste.
    Pour choisir une langue, appuyer sur 'Entrée' ou 'Espace'. Si la langue choisie est différente de la langue en cours, une fenêtre de confirmation s'ouvre. Cette fenêtre peut être fermée en appuyant sur la touche 'Echap'.

  • Utiliser la touche 'Tab' pour naviguer entre les en-têtes.
    Lorsque le focus est positionné sur un en-tête, appuyer sur 'Entrée' ou 'Espace' pour dérouler le panneau correspondant (lorsqu'un nouveau panneau s'ouvre, les autres se ferment automatiquement).

  • Quand le focus est positionné sur une image, les touches 'Flèche gauche' et 'Flèche droite' permettent d'afficher respectivement l'image précédente et l'image suivante.
    Quand le focus est positionné dans la barre de controle, les touches 'Flèche gauche' et 'flèche droite' permettent de naviguer à l'intérieur et d'afficher automatiquement les images correspondantes.

  • Utiliser 'Tab' pour naviguer dans la barre de controle.
    Le fonctionnement des potentiomètres du site est détaillé plus haut dans cette page.
    Appuyer sur la touche 'Echap' pour quitter le mode plein écran.
    Appuyer sur 'Espace' pour lancer ou mettre en pause la vidéo.
