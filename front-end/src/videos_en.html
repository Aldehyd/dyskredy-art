Keyboard Shortcuts

This website contains complex components. You will find descriptions about their behaviors and how to use them with the keyboard below.

How to navigate in this window with keyboard ?

Informations are contained in an accordion component.
Use 'Tab' to navigate between headers.
When a header is focused, use 'Enter' or 'Space' to open its panel (when a new panel opens, all the other panels automatically close).
Press 'Escape' key to close this window.

Note : you can open this window from any page of the website by simultaneously press 'Control' + 'Alt' + 'S'.

  • Modal windows can be closed by pressing 'Escape' key.
    Focus is trapped inside the window and wrapping focusable elements from the last to the first.

  • Use arrow keys to set the slider value.
    'Arrow up' and 'Arrow right' increase the value.
    'Arrow down' and 'Arrow left' decrease the value.
    'Page up' increase the value faster than arrow keys.
    'Page down' decrease the value faster than arrow keys.
    'End' set the slider on its highest value.
    'Home' set the slider on its lowest value.

  • 'Enter' or 'Escape' open the page related to the focused menu button.
    Use arrow keys to navigate inside the main menu.
    'Arrow right' and 'Arrow left' allow you to navigate between main buttons, wrapping from the last to the first.
    Some of the main buttons contain a submenu. To open a submenu, press 'Arrow down' or 'Arrow up' when focusing its main button parent.
    'Arrow down' open submenu and focus its first item.
    'Arrow up' open submenu and focus its last item.

    Use 'Arrow up' and 'Arrow down' to navigate inside a submenu, wrapping links from last to first.
    By default, submenus do not open by focusing main buttons.
    However, when a submenu has once been opened, all the submenus automatically open when focusing their main button parent.
    To cancel this behavior, close a submenu by pressing 'Escape' key.
    Pressing letter key allows you to focus the first button beginning by this letter.
    For example, to reach 'Contact' button, press 'C'.
    You can do the same inside submenus.

  • 'Enter', 'Space', 'Arrow down' and 'Arrow up' keys open languages menu.
    When the menu is closed, 'Arrow down' open it and focus its first item.
    When the menu is closed, 'Arrow up' open it and focus its last item.
    'Escape' closes the menu.
    'Arrow up' and 'Arrow down' allow you to navigate inside the menu, wrapping items from last to first.
    Chose a language by pressing 'Enter' or 'Space' keys. If the chosen language is different from the current language, a modal window opens. You can close it (and cancel language choice) by pressing 'Escape'.

  • Use 'Tab' key to navigate between headers.
    When focusing a header, use 'Enter' or 'Space' key to open its panel (when a new panel opens, all the other panels automatically close).

  • Slide show is a modal window. Modal window behavior is described above.
    When focusing a picture, 'Arrow left' and 'Arrow right' show respectively previous and next picture.
    When focus is inside the control bar, use 'Arrow left' and 'Arrow right' to navigate and automatically show pictures.

  • Use 'Tab' to navigate inside the control bar.
    Slide behavior is described above.
    Press 'Escape' key to quit full screen view.
    When focusing the video, press 'Space' key to play or pause the video.
